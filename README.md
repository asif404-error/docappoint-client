# DocAppoint — Doctor Appointment Booking System

**Live Site:** Live Site will be visible here after the deployment

## About DocAppoint

DocAppoint is a modern and fully responsive doctor appointment booking platform built with Next.js, Node.js, Express, and MongoDB. Users can browse top-rated doctors, view details, and book appointments instantly with secure authentication.

## Features

**Secure Authentication System** — Users can register and login using email/password or Google OAuth powered by Better Auth with JWT token-based security to protect private routes and API endpoints.

**Browse & Book Doctors** — Users can browse all available doctors, search by doctor name, sort by rating, fee or experience, view complete doctor details and book appointments instantly with a smooth booking modal.

**Manage Appointments** — Logged-in users can view all their booked appointments in the dashboard, update appointment details with a pre-filled modal and delete appointments instantly with real-time UI updates.

**Doctor Reviews System** — Patients can leave reviews and star ratings for doctors after their appointment, helping other users make informed decisions when choosing a doctor.

**Dark & Light Theme Toggle** — The application supports a fully functional dark and light mode toggle that persists across page refreshes using localStorage, providing a comfortable viewing experience for all users.

**Fully Responsive Design** — The entire application is optimized for all screen sizes including mobile, tablet and desktop with a clean and modern UI using Tailwind CSS and DaisyUI.

**User Profile Management** — Users can view and update their profile information including name and profile photo directly from the dashboard without any page refresh.

## Tech Stack

**Frontend:** Next.js, Tailwind CSS, DaisyUI
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Authentication:** Better Auth, JWT
**Others:** Swiper JS, React Hot Toast, React Icons
